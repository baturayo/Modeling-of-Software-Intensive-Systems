Readme

All the models can be found in the Models subdirectory.

To run the experiment.py file, plotpy and tqdm have to be installed.
These libraries can be installed with:
sudo pip install plotpy
sudo pip install tqdm

Running the experiment.py file can take a long time because of the amount of simulations ran to get significant data.
To get a quicker view of the results, you can change the numtrains on line 30 to a lower amount(50 for example)
